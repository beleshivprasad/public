Program for Celsius To Fahrenheit conversion

Problem Statement: Write a code in any programming language that converts Centigrade to Fahrenheit.

#include  
using namespace std; 
float Cel_To_Fah(float n) 
{ 
	return ((n * 9.0 / 5.0) + 32.0); 
} 
int main() 
{ 
	float n = 20.0; 
	cout << Cel_To_Fah(n); 
	return 0; 
}
Sample Input
0
Sample Output
32

Formula for converting Celsius scale to Fahrenheit scale

T(°F) = T(°C) × 9/5 + 32

Variables used in this program

n

Time Complexity

log(n2)
O(1)
